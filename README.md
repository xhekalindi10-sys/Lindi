# Ndertimi i nje website per agjensi turistike
* Ky eshte nje projekt individual qe eshte ndertuar per krijimin e nje websiti rreth nje agjensie turistike te ideuar nga une.
* Ky projekt permban imazhe,file Html,Css,Javascript dhe elemente te Bootstrap qe bejne te mundur paraqitjen e websitit.
* Filet Html jane perdorur për të strukturuar dhe paraqitur përmbajtjen e websitit.
* File-i CSS përdoret për të stiluar dhe dizajnuar faqet HTML. Ndryshe nga HTML që siguron strukturën, CSS përcakton pamjen vizuale.
* Ndersa file-t JavaScript përdoren për të shtuar funksionalitet dhe interaktivitet ne faqe.
